# How to Run This Project

## Prerequisites
- Node.js & npm installed
- At least 500MB of free disk space

## Steps to Run the Project

1. **Free up disk space**
   - Delete unnecessary files or programs
   - Empty the recycle bin
   - Clear browser cache and temporary files

2. **Navigate to the project directory**
   ```bash
   cd c:\Users\khair\Downloads\azizi-digital-canvas-main\azizi-digital-canvas-main
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Access the application**
   Once the server starts successfully, open your browser and go to:
   http://localhost:5173

## Troubleshooting

If you encounter any issues:
- Ensure Node.js is properly installed by running `node --version`
- Clear npm cache with `npm cache clean --force`
- Delete node_modules folder and package-lock.json, then reinstall
